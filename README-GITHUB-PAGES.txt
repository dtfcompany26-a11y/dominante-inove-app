COMO PUBLICAR NO GITHUB PAGES

Link final esperado:
https://SEU-USUARIO.github.io/dominante-inove-app/

Passos:

1. Entre em https://github.com/new
2. Crie um repositorio publico chamado:
   dominante-inove-app
3. Envie estes arquivos para a raiz do repositorio:
   - index.html
   - styles.css
   - script.js
   - .nojekyll
4. No GitHub, abra:
   Settings > Pages
5. Em Build and deployment:
   Source: Deploy from a branch
   Branch: main
   Folder: /root
6. Salve e aguarde alguns minutos.

Depois disso, o app abre pelo celular sem depender deste computador.
